# research-automation-

This repository contains my research automation workflow deployed on a Hetzner VM.

## Overview

The project is used to automate research collection, report generation, and notification workflows.

## Main folders

- `apps/`: application scripts and workflow entry points
- `config/`: configuration files
- `data/`: local working data
- `research-agent-platform/`: research automation platform code and scripts

## Environment

Copy `.env.example` and fill in the required environment variables before running workflows.

## Git workflow

This project is connected to GitHub through SSH from the Hetzner VM.
